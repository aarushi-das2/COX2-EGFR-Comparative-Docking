# Comparative Computational Analysis of Plant and Endophyte-Derived Metabolites Targeting COX-2 and EGFR

This repository contains the processed datasets and analysis scripts used in the manuscript.

## Contents
- PlantScreening.xlsx: Docking scores, top hits, ADMET summary, and protein–ligand interaction data.
- tanimoto_similarity.ipynb: RDKit-based structural similarity analysis.
- pca_analysis.ipynb: Principal component analysis of molecular descriptors.
- docking_statistics.ipynb: Statistical comparison of docking scores.

Software used:
- AutoDock Vina
- PyMOL
- BIOVIA Discovery Studio Visualizer
- SwissADME
- RDKit
- Python
- STRING
